## ðŸ“˜ Lecture 2 â€“ Numerical Differentiation

### Learning Outcomes
- Compute derivatives using high-accuracy finite difference formulas  
- Apply Richardson extrapolation for improved derivative estimates  
- Handle derivatives of unequally spaced and noisy data  
- Compute partial derivatives and integrate with numerical integration  

### Lecture Flow
1. **High-Accuracy Differentiation Formulas (25 min)**  
   - Forward, backward, and central differences  
   - Example: \( f(x) = e^x \)  

2. **Richardson Extrapolation (20 min)**  
   - Refined derivative estimates using step size variation  
   - Example: Central difference improvement  

3. **Unequally Spaced Data (15 min)**  
   - Adjusted finite difference formulas  
   - Application: experimental measurements  

4. **Derivatives and Integrals with Errors (15 min)**  
   - Handling measurement noise in computations  
   - Error propagation in combined differentiation and integration  

5. **Partial Derivatives (15 min)**  
   - Finite difference formulas for \( f(x,y) \)  
   - Example: Temperature or pressure field gradients  

6. **Numerical Integration/Differentiation Balance (10 min)**  
   - Combining differentiation and integration in engineering analysis  
   - Example: Velocity from noisy position data
