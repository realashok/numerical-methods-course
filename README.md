## How to Use
Clone the repo and explore the `lectures/` and `labs/` directories.

```bash
git clone https://github.com/realashok/numerical-methods-course.git
cd numerical-methods-course
pip install -r requirements.txt
python labs/root_finding/newton_raphson.py
```
